## ADDED Requirements

### Requirement: 系统必须提供资讯内容最小管理与展示能力
系统 MUST 支持资讯内容在后台创建与发布，并支持小程序按发布时间查询列表。

#### Scenario: 发布资讯并在小程序展示
- **WHEN** 管理员发布一条资讯
- **THEN** 小程序资讯列表接口可查询到该条已发布资讯

### Requirement: 系统必须提供订单能力基础接口占位
系统 MUST 支持创建订单记录、查询订单状态、更新订单状态的基础接口，不要求真实支付扣款。

#### Scenario: 创建订单记录
- **WHEN** 用户触发付费入口并提交商品标识
- **THEN** 系统生成待支付订单并返回订单编号与状态

#### Scenario: 查询订单状态
- **WHEN** 客户端按订单编号查询
- **THEN** 系统返回当前订单状态与最近更新时间
