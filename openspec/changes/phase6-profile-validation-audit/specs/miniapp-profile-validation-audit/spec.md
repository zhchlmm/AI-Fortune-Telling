## ADDED Requirements

### Requirement: 小程序资料更新必须执行字段校验
系统 MUST 在保存资料时校验昵称、邮箱、手机号格式，并允许字段为空。

#### Scenario: 邮箱格式不合法
- **WHEN** 用户提交非空且格式非法的邮箱
- **THEN** 系统返回 400 并包含错误信息

#### Scenario: 手机号为空
- **WHEN** 用户提交空手机号
- **THEN** 系统接受请求并保存为 null

### Requirement: 资料更新必须记录审计日志
系统 MUST 对每次资料更新写入审计记录，且敏感字段脱敏。

#### Scenario: 更新手机号成功
- **WHEN** 用户成功更新手机号
- **THEN** 审计日志记录变更事件且不存储完整明文手机号
