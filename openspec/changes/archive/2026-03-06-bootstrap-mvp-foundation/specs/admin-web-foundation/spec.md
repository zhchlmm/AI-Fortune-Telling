## ADDED Requirements

### Requirement: 管理后台必须提供基础登录壳层与受保护路由
系统 MUST 提供登录页、基础布局页及受保护路由机制，未登录用户不得访问管理页面。

#### Scenario: 未登录访问管理页
- **WHEN** 未登录用户直接访问管理路由
- **THEN** 系统重定向至登录页

### Requirement: 管理后台必须提供核心运营页面骨架
系统 MUST 提供仪表盘、占卜模板管理页、内容资讯管理页、会话查询页的最小页面骨架。

#### Scenario: 管理员进入模板管理页
- **WHEN** 管理员通过导航进入模板管理
- **THEN** 系统展示模板列表占位与新增/编辑操作入口
