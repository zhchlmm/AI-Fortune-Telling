## ADDED Requirements

### Requirement: 后台API必须提供基础健康检查与版本化接口前缀
系统 MUST 暴露健康检查接口与统一版本化 API 前缀，供小程序与后台前端接入。

#### Scenario: 前端探活接口
- **WHEN** 客户端调用健康检查接口
- **THEN** 系统返回可用状态与服务时间戳

### Requirement: 后台API必须提供占卜会话最小闭环接口
系统 MUST 提供创建占卜会话、按ID查询会话、按用户查询历史会话列表接口。

#### Scenario: 创建占卜会话成功
- **WHEN** 客户端提交有效占卜参数（类型、出生信息或星座参数）
- **THEN** 系统创建会话并返回会话ID、结果摘要与创建时间

#### Scenario: 查询历史会话成功
- **WHEN** 客户端提交用户标识与分页参数
- **THEN** 系统返回按时间倒序的会话列表
